使い方

環境構築編

  1. 環境を整えます
  2. リポジトリからクローンします
    3. git clone git@github.com:sapodeveloper/sapodeveloper.middleman.git sapodeveloper.github.io
  3. リポジトリの中へカレントディレクトリを変更します
    4. cd sapodeveloper.github.io
  4. 環境設定コマンドを実行します
    5. rake setup
  5. 記事を書きましょう
注意

クローン元とデプロイ先は違うリポジトリです。
middlemanを保存しているのはsapodeveloper.middlemanリポジトリ
デプロイ時に生成するサイト構成を保存しているのはsapodeveloper.github.ioリポジトリ


記事の書き方編

  1. rake new_post[記事の名前(英語で)]
    2. 2013年11月12日に作成した場合、 source/blog/2013/11 に2013-11-12-記事の名前.html.erb が作られます。
  2. 作られたファイルのFrontmatterを編集します
    3. ---
    title: "記事の名前(ここは日本語でもOK)"
    date: 作成した日付が自動で入ります(例：2013-11-12)
    tags: タグ付けができます(例：Ruby, Ruby on Rails 複数有る場合は","で区切ります)
    auther: 誰が作成したか分かるように名前を書きます
    ---
  3. Frontmatterの下から記事を書きます。markdownですが、markupで書けます
  4. 書いている記事はrake previewを実行し、localhost:4567にアクセスして確認できます
  5. 記事が書けたらcommitします
  6. commit後にrake deployでsapodeveloper.github.ioにデプロイします

コードの書き方編

記事等でコードを書きたいときは下記のように書くとシンタックスハイライトされます。
